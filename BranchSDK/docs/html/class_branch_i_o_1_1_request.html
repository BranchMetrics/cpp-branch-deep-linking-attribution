◆ getBackoffMillis()

int32_t BranchIO::Request::getBackoffMillis ( ) const
protected

Get the number of milliseconds to wait until the next request, based on the attempt count. Returns 8000, 16000, 32000, 64000, then 120000 for all successive attempts.

Note that Poco::Thread::sleep() takes a long, like Poco::Condition::wait and friends. But long is different sizes depending on the architecture. The linter wants a specific-size type here. Using int32_t avoids compiler warnings/errors on 64-bit architectures.

Returns
a delay in ms
